# Project Features (v1.0)

## Core Gameplay
- **Deck System**: Fixed deck of scenario cards for consistent gameplay (`GameEngine.jsx`).
- **Swipe Mechanics**: Tinder-style swipe interface (`SwipeCard.jsx`) with "Approve" (Right) and "Reject" (Left) actions.
- **Resource Management**: Real-time tracking of:
  - **Budget** & **Land** (Finite resources)
  - **Health** & **Happiness** (City stats)
  - Visualized via `HUD.jsx`.
- **Dynamic Backgrounds**: `CityBackground.jsx` likely provides visual context.

## Input & Engagement
- **Press Conference**: Special cards (`PressConferenceCard.jsx`) for collecting qualitative user suggestions.
- **Start Screen**: Nickname entry ("Simple Authentication") (`HomeScreen.jsx`).
- **Impact Ledger**: End-game summary showing user choices and impact (`ImpactLedger.jsx`).
- **Leaderboard**: Competitive element for tracking high scores (`Leaderboard.jsx`).

## Administration
- **Admin Dashboard**: Planner view (`AdminDashboard.jsx`) to review submitted suggestions and swipe statistics.
